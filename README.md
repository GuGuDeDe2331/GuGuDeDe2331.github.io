# The Human Signal

An immersive single-page digital artwork for GitHub Pages.

This site is not a template. It is a small browser-based installation about the long signal of human civilization: fire, language, number, cities, compassion, machines, memory, and the sky.

## Local Preview

```bash
python -m http.server 8010
```

Then open:

```text
http://127.0.0.1:8010/
```

## Files

- `index.html`: the artwork shell and poetic layers
- `assets/css/styles.css`: visual composition and responsive layout
- `assets/js/art.js`: Three.js scene, interaction, fallback canvas, and scroll choreography
